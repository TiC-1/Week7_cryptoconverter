  - Express http://expressjs.com/

  - Handlebars https://handlebarsjs.com/
  - Supertest https://github.com/visionmedia/supertest
- Font-end
  - Vue https://vuejs.org/
  - QUnit https://qunitjs.com/
- Test
  - Travis https://github.com/dwyl/learn-travis
